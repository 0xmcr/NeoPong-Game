# Security Policy

We take security seriously at AeroCut AI. If you discover a vulnerability in this project, please follow the guidelines below to report it.

## Supported Versions

Only the current main branch receives security updates. Please ensure you are running the latest version of the codebase.

| Version | Supported |
| :--- | :--- |
| Latest (main branch) | 😄 Yes |
| All older versions | ❌ No |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues. Instead, notify us privately by contacting the project owners or maintainers directly.

Once a vulnerability report is received:
1.  We will acknowledge your report promptly and start validating the issue.
2.  We will work on a patch or fix to resolve the vulnerability.
3.  Once the fix is released, we will publish an advisory detailing the vulnerability and its remediation.

Thank you for helping keep AeroCut AI secure!
